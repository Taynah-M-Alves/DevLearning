Objetivo

Neste nível, seu objetivo é aprender a criar um card de produto responsivo, que se adapta a diferentes tamanhos de tela e se organiza de forma agradável dentro de um container com múltiplos cards.

O que é um Card de Produtos?

Um card de produto é uma pequena seção visual em um site ou e-commerce que exibe informações básicas sobre um produto, como imagem, título, descrição e preço. Ele é usado para chamar a atenção do usuário e incentivá-lo a realizar uma compra ou interagir com o produto de alguma forma.

Como Criar o Card de Produto

Um card de produto é uma pequena seção visual em um site ou e-commerce que exibe informações básicas sobre um produto, como imagem, título, descrição e preço. Ele é usado para chamar a atenção do usuário e incentivá-lo a realizar uma compra ou interagir com o produto de alguma forma.

Passo 1: Criar a Estrutura HTML do Card

A estrutura HTML do card é bastante simples. Vamos criar uma <div> para representar o card, e dentro dela vamos colocar as informações do produto. Aqui está o código básico:

          
            <div class="card">
              <img src="imagem-do-produto.jpg" alt="Imagem do Produto">
              <h2>Nome do Produto</h2>
              <p>Descrição do produto</p>
              <div class="preco">R$ 99,99</div>
              <button>Comprar</button>
            </div>

Passo 2: Estilizando o Card com CSS

Agora vamos adicionar estilos para que o card fique bonito e organizado. No CSS, vamos trabalhar com o fundo, o tamanho, o sombreamento e o alinhamento dos itens. Aqui está como você pode fazer isso:

            .card {
              background-color: #fff; /* Cor de fundo branca */ 
              box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); /* Sombra ao redor do card */
              border-radius: 10px; /* Bordas arredondadas */
              padding: 20px; /* Espaçamento interno */
              text-align: center; /* Alinha os itens no centro */
              width: 280px; /* Largura do card */
              height: fit-content; /* Altura ajustada ao conteúdo */
              display: flex; /* Flexbox para organizar os itens */
              flex-direction: column; /* Coloca os itens na vertical */
              justify-content: center; /* Centraliza verticalmente */
              align-items: center; /* Centraliza horizontalmente */
            }

Com esse estilo, estamos dizendo ao navegador para criar um card com fundo branco, bordas arredondadas, sombra e centralização dos conteúdos. Usamos flexbox para facilitar o alinhamento e organização dos itens dentro do card.

Passo 3: Estilizando a Imagem

Para a imagem do produto, queremos que ela tenha uma largura ajustada e bordas arredondadas. Aqui está o código para isso:

            .card img {
              max-width: 40%; /* A imagem ocupará no máximo 40% da largura do card */
              height: auto; /* Mantém a proporção da imagem */
              border-radius: 10px; /* Bordas arredondadas na imagem */
              margin-top: 15px; /* Espaçamento superior */
            }

Este estilo faz com que a imagem fique dentro do card e tenha bordas arredondadas, além de ajustar seu tamanho automaticamente para não ultrapassar 40% da largura do card.

Passo 4: Estilizando o Texto do Card

Agora vamos adicionar estilo ao nome, descrição e preço do produto:

          
            .card h2 {
              color: #222; /* Cor do texto do nome do produto */
              margin-top: 20px; /* Espaçamento superior */
              margin-bottom: 10px; /* Espaçamento inferior */
              width: 70%; /* A largura do nome será de 70% do card */
            }
      
            .card p {
              color: #222; /* Cor da descrição */
              font-size: 16px; /* Tamanho da fonte da descrição */
              margin-bottom: 20px; /* Espaçamento inferior */
            }
      
            .card .preco {
              color: #FF0000; /* Cor vermelha para o preço */
              font-weight: bold; /* Deixa o preço em negrito */
              font-size: 18px; /* Tamanho maior para o preço */
              margin-bottom: 20px; /* Espaçamento inferior */
            }

Agora o nome, a descrição e o preço do produto estão estilizados de forma adequada para destacar cada um.

Como criar uma área de card's

Uma área de card's ou Card-box é o conjunto de vários card's dentro de um container. Costuma ser usado quando a empresa precisa apresentar mais de um produto ou serviço.

Passo 2: Criando um Layout Responsivo

Para garantir que os cards fiquem bonitos em telas menores, vamos usar o CSS Flexbox para criar um layout responsivo. Flexbox permite que os cards se ajustem ao tamanho da tela e se reorganizem conforme necessário.

            .card-container {
              display: flex;
              flex-wrap: wrap; /* Permite que os cards se movam para uma nova linha se não houver espaço */
              justify-content: space-evenly; /* Espaçamento igual entre os cards */
            }

Agora, os cards ficarão organizados em linhas e se ajustam automaticamente dependendo do tamanho da tela.

Passo 3: Tornando o Layout Adaptável

Com o Flexbox ativado, a próxima etapa é garantir que o layout dos cards se ajuste em dispositivos com diferentes tamanhos de tela. Aqui, vamos controlar o número de cards por linha em diferentes tamanhos de tela, usando media queries:

            @media screen and (max-width: 1200px) {
              .card-container {
                flex-wrap: wrap;
              }
              .card {
                width: 45%; /* Para telas grandes, dois cards por linha */
              }
            }

            @media screen and (max-width: 768px) {
              .card {
                width: 100%; /* Para telas médias, um card por linha */
              }
            }

Com as media queries, os cards agora são responsivos e mudam de layout conforme o tamanho da tela.
