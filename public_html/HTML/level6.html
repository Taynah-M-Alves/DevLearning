OBJETIVO

Neste nível, seu objetivo é aprender a criar um rodapé informativo e responsivo, que se adapta a diferentes tamanhos de tela, incluindo links úteis, informações de contato e redes sociais integradas.

O que é um RODAPÉ?

Um rodapé é uma seção na parte inferior de uma página web que contém informações complementares, como links para políticas do site, informações de contato, e links para redes sociais. É crucial para a navegação e fornece informações úteis aos usuários.

Como Criar um Rodapé Responsivo

Passo 1: Estrutura HTML

Vamos começar criando a estrutura básica do rodapé. Usamos a tag <footer> para semântica e acessibilidade. Dentro dela, colocamos um <div> com a classe "container" para agrupar e organizar as informações.

Motivo: A estrutura semântica ajuda motores de busca e leitores de tela a entenderem a função do rodapé.

<footer>
  <div class="container">
    <p>Email: contato@devlearning.com</p> 
    <p>Telefone: (11) 98765-4321</p> 
  </div>
</footer>

Passo 2: Links Úteis

Adicionamos links para páginas importantes como "Política de Privacidade" e "Termos de Uso". Usamos uma lista não ordenada <ul> para manter a organização e a estrutura.

Motivo: Listas ajudam na acessibilidade e permitem fácil estilização.

<ul>
  <li><a href="privacidade.html">Política de Privacidade</a></li> 
  <li><a href="termos.html">Termos de Uso</a></li> 
</ul>

Passo 3: Contato

Inserimos informações de contato importantes, como telefone e redes sociais, para facilitar a comunicação com os usuários.

Motivo: Ter múltiplas formas de contato aumenta a acessibilidade e a confiança dos usuários.

<div class="contact-info">
  <p>WhatsApp: (11) 91234-5678</p> 
  <p>Instagram: @devlearning</p> 
</div>

Passo 4: Estilização com CSS

Aplicamos CSS para estilizar o rodapé, ajustando cores, espaçamentos e fontes. Usamos flexbox para uma distribuição uniforme dos elementos.

Motivo: Um design coerente e organizado melhora a experiência do usuário e a usabilidade.

footer {
  background-color: #333; /* Cor de fundo do rodapé */
  color: white; /* Cor do texto */
  display: flex; /* Usando flexbox para layout */
  justify-content: space-around; /* Distribuição uniforme */
  padding: 20px; /* Espaçamento interno */
}

Passo 5: Tornando Responsivo

Usamos media queries para ajustar o layout em telas menores. Isso garante que o rodapé seja fácil de ler e interagir em dispositivos móveis.

Motivo: A responsividade é essencial para a acessibilidade e usabilidade em diversos dispositivos.

@media (max-width: 600px) {
  footer {
    flex-direction: column; /* Empilhar elementos verticalmente */
    text-align: center; /* Centralizar texto */
  }
}

Passo 6: Teste e Iteração

Teste o rodapé em diferentes navegadores e dispositivos. Use ferramentas como Lighthouse para verificar a performance e acessibilidade.

Motivo: Testar garante que o rodapé funcione corretamente para todos os usuários.

/* Teste o site e faça ajustes */

Estrutura do Rodapé Utilizada

Este rodapé foi construído com os seguintes elementos:

  • <footer>: A estrutura principal do rodapé.
  • <div class="container">: Um contêiner que agrupa todas as informações.
  • <h3>: O título para a seção de contato.
  • <p>: Parágrafos que contêm informações de contato e redes sociais.
