A seção "Sobre Nós" é um bloco de conteúdo destinado a apresentar informações sobre o propósito e os valores do DevLearning. A estrutura e o conteúdo dessa seção foram organizados para:

  • Descrição do Propósito: Explica o motivo da criação do DevLearning, que é ajudar iniciantes a desenvolver habilidades básicas de HTML e CSS.
  • Apresentação dos Valores: Exibe os valores fundamentais da iniciativa, como acessibilidade, inovação, empatia e compartilhamento de conhecimento.
  • Design e Organização: Utiliza um layout flexível para dividir o texto e imagens (se adicionadas) de forma equilibrada.
  • Empatia: Sabemos que a jornada de aprendizado pode ser difícil e estamos aqui para ajudar.

O CSS responsivo aplicado na seção "Sobre Nós" é projetado para garantir que o layout se adapte a diferentes tamanhos de tela, proporcionando uma boa experiência em dispositivos móveis e desktops. As principais características são:

Layout Flexível:

  • A classe .about-content utiliza o Flexbox para alinhar e organizar o conteúdo horizontalmente em telas maiores.
  • Com o atributo flex-wrap, os elementos dentro dessa seção podem ser reorganizados para caber em espaços menores.

Adaptação em Telas Menores:

  • Usando media queries (@media), o layout muda para uma estrutura vertical em telas menores, com os elementos empilhados um sobre o outro.
  • Para dispositivos com largura abaixo de 768px, a orientação muda de horizontal para vertical com a propriedade flex-direction: column.
  • Para dispositivos ainda menores, abaixo de 480px, os tamanhos de fonte e margens são reduzidos para melhorar a legibilidade.

Estilos Dinâmicos:

  • Tamanhos de fonte e espaçamentos são ajustados para evitar que textos fiquem muito pequenos ou muito apertados.
  • As imagens, caso usadas, são redimensionadas automaticamente com width: 100% e height: auto, garantindo que não ultrapassem os limites da tela.

Sobre o DevLearning

O DevLearning foi criado em 2024 por quatro estudantes de Engenharia de Software com o objetivo de ajudar iniciantes a aprenderem como criar sua primeira landing page usando HTML e CSS.

A ideia surgiu após enfrentarmos desafios semelhantes no primeiro semestre da faculdade, ao tentarmos criar nosso primeiro site. Decidimos compartilhar nossa experiência e ajudar outras pessoas a superarem essas dificuldades de forma prática e acessível.

Valores do DevLearning

  • Acessibilidade: Tornamos o aprendizado fácil e acessível para todos.
  • Inovação: Estamos sempre atualizados com as últimas tendências de web design.
  • Compartilhamento de Conhecimento: Acreditamos no poder do aprendizado colaborativo.
  • Empatia: Sabemos que a jornada de aprendizado pode ser difícil e estamos aqui para ajudar.
